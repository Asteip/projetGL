Projet Génie Logiciel M1ALMA 2016.


Groupe : Alexis Bonnin, Joachim Clayon, Esteban Launey, Sacha Loriot

Sujet :
Le sujet de ce projet est de réaliser la conception d'une application logiciel pour l'emission et la gestion de souhaits pédagogique émis et 
gérés par l'équipe enseignante d'une université.
